# RDynLib 1.0 
**RDynLib 1.0  (this repository) is no longer under active development.**\
This repository contains the original version of RDynLib, including the R functions used to create and expand an annotated spectral database from in-house LC-MS/MS and LC-MSn experiments.\
Ongoing development of **RDynLib 2.0** has moved to a new repository: https://github.com/rebdau/RDynLib_SQL.
